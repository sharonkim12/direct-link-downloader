How to use

  • 1) Paste the link you’re about to click.
  • 2) Click Analyze. You’ll be sent to the result page.
  • 3) If it returns text/html, it’s usually a fake download page.
  • 4) If it returns a file type + size, it’s likely a real file.
Supported: mp4, zip, pdf, exe (direct links).
Not supported: YouTube, most streaming pages.
